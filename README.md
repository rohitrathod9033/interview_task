# Fullstack React + Node.js App

This is a fullstack application with a React + Vite frontend and a Node.js + Express backend.

## Structure
- `frontend/` - React application
- `backend/` - Node.js Express server

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run dev
```
